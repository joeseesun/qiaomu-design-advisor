# 交付前门禁（Pre-flight Checklist）

> 来源：taste-skill 的 pre-flight 机制 + 受控实验全部发现。
> 规则：交付任何页面/组件前逐条过。**任何一条不过就不交付**，修完再查一遍。

## 第 0 步 · 用户偏好账本（最高优先级）

- [ ] 已读取 `references/user-preferences.md`，逐条对照所有生效规则
- [ ] 本次会话中用户新给的反馈已按自进化机制入库，且已用新规则复查本次全部产物
- [ ] 自己在截图、浏览器验收、回传测试、lint/test/build 或真实交互里发现的可复现问题，
      已自动写入设计记忆并同步为规范/门禁；当前产物已复查修正

## A. 反套路扫描（AI Tells）

- [ ] 无紫蓝渐变、无 `#000000` 纯黑、无外发光霓虹、无大标题渐变文字
- [ ] 字体不是 Inter/Roboto/Arial/系统默认，且和上次生成不同
- [ ] **全页无 `font-style: italic`**（中文伪斜体尤其禁止；强调只用字重/颜色/字号）
- [ ] Hero 不是居中构图（除非设计读取明确要求）
- [ ] 无三等分卡片横排；卡片没有被当作万能分组容器
- [ ] 无 John Doe/张三、无 99.99% 式整数假数据、无 Acme 式烂梗名
- [ ] 文案无"革命性/颠覆/无缝/赋能/下一代"；感叹号 ≤ 1
- [ ] 眉批（eyebrow）数量 ≤ ⌈section 数 / 3⌉；无 "SECTION 01" 式编号 meta-label
- [ ] H1 不超过 2-3 行（窄容器多行大标题 = 失败）
- [ ] Bento/网格无空洞（`grid-auto-flow: dense`，span 完全咬合）

## A2. 进攻三步核验（craft-loop.md）

- [ ] DNA 注入：能逐条说出"从 {site} 的 DESIGN.md 偷了哪 3-5 个具体值"
- [ ] 工艺密度：签名动作 ≥ 5 项，可逐项指认位置
- [ ] 两轮制：第一轮截图 + 四视角评审记录（各至少一处）+ 第二轮修复均已发生

## A3. 资产与可读性（assets-and-readability.md）

- [ ] Hero 视觉侧过关：图像生成资产 / 高保真组件预览（深度≥3层、元素≥8、有活细节、
      构图有姿态）/ 抽象编辑构图，三选一——无空心 div 假截图、无"文字+渐变斑点"
- [ ] 暗底正文 `rgba(255,255,255,.85)`、次要文字 ≥ `.60`；正文区纹理 ≤ 3%
- [ ] 每个 CTA 做过对比度审计；主 CTA 不换行；同一意图全页一个文案
- [ ] 文案自审完成：无语法不通/指代不明/AI 装深沉句；假精确数据已清理
- [ ] 多方向展示（如适用）：每方向全宽沉浸带 + 独立工艺密度 ≥ 3 + 设计宣言入版面

## A4. 功能契约（任何 VARIANCE 值都不豁免）

- [ ] 已写下本页功能契约（存在理由 / 3 秒必得信息 / 必须能完成的动作）并逐条核对
- [ ] 页面类型的核心标识首屏可辨（404 的错误码、定价的价格、表单的提交路径）
- [ ] 创意元素没有遮蔽或稀释任何契约项

## A5. 方案预览交付门（Phase 2 必查）

- [ ] 如果用户要求设计方案、重构方案、多个方向或风格建议，已生成可打开的浅层预览目录 `design-previews/YYYY-MM-DD-任务名/index.html`
- [ ] `index.html` 是自包含 HTML，并通过 HTML 解析或等价结构检查
- [ ] 预览目录非隐藏、层级不超过 `design-previews/<date-task>/`；用户可直接找到、打包、分享
- [ ] 预览页用低干扰单句说明"这是设计方向样机 / 不是最终 App 或最终页面 / 选择后才进入正式实现"
- [ ] 预览页包含至少 4 个真实 mini mockup，且差异覆盖布局、密度、色彩或字体中的多个轴，不只是换颜色
- [ ] 多方向已按 subagent 或独立 brief 隔离生成；若无法使用 subagent，已说明降级原因
- [ ] 每个方向使用独立 stage 与 scoped CSS / 独立类名前缀，无全局样式污染其它方向
- [ ] 桌面和移动端截图都已检查：页面无横向滚动，mini mockup 占满或居中，内部文字/控件没有互相挤压或遮挡
- [ ] 设计样机区和方向说明区已分离；主视觉区域只放 mockup、短标签和选择按钮，不混入长解释、参数表或评审说明
- [ ] 预览页遵守奥卡姆剃刀：无重复说明、小字教程、长段免责声明或系统自言自语
- [ ] 其中一个方向明确标注「推荐」，并在页面和回复中说明推荐理由
- [ ] 已启动本地预览回传服务并自动打开浏览器，或明确说明服务无法启动的降级原因
- [ ] 每次重启预览服务后已明确告知新的本地 URL；旧端口页面视为过期缓存，不作为验收证据
- [ ] 预览页外侧选择 UI 保持统一：固定顶部状态栏、快捷键提示、统一选中高亮与回传 toast
- [ ] 选择按钮已有"选择 A/B/C/D · 方向名"文案时，顶部或卡片上方没有重复"点按钮选择"说明
- [ ] 每个方向卡片都有明显的选择按钮，按钮文本包含方向字母和方向名
- [ ] 运行本地预览服务后，每个方向只有一组选择按钮；HTML 自带按钮已用 `.qmdp-pick-button`
      或等价标识避免服务重复注入
- [ ] HTML 自带选择按钮已有自己的回传处理时，预览服务不会再次处理同一次点击；服务自动注入按钮必须带
      `data-qmdp-injected` 或等价标识
- [ ] 重复按钮检查基于服务注入后的真实 DOM / 截图，而不只是源码 grep；嵌套在 `.meta`
      等内部容器的按钮也必须被服务识别
- [ ] 对用户展示三拨盘时使用中文标签和解释，说明用户可调整；没有只显示
      `VARIANCE/MOTION/DENSITY` 英文参数名
- [ ] 预览服务提供 `POST /api/select` 与 `GET /api/selection`，选择会写入预览目录内的 `selection.json`
- [ ] 用户可见的回传文案只说"已回传"或"当前未能回传"，没有绑定具体工具名或调用端
- [ ] watcher 优先使用文件事件或等价即时通知，短轮询兜底间隔不超过 250ms
- [ ] 执行代理进入 Phase 3 前已观察到 `selection.json` 或 `QIAOMU_DESIGN_SELECTION::` 日志，或用户在对话中明确选择/授权默认
- [ ] 启动预览后没有发送 final 结束回合；已保持 watcher/轮询，直到选择被消费或用户明确说稍后再选
- [ ] 最终回复给出可点击的本地文件路径和本地预览 URL；缺少路径不得交付
- [ ] 若因权限或项目约束无法写文件，已明确说明阻塞原因，并提供完整可保存 HTML 内容作为替代

## A6. 乔木身份边界

- [ ] 除非用户明确要求，生成页面没有默认添加打赏二维码、公众号关注、GitHub/X 浮条、乔木推荐入口或个人 Profile 区块
- [ ] 对外分享页面如需署名，最多使用低干扰页脚 `Powered by 向阳乔木` 链接到 `https://qiaomu.ai/`

## B. 设计读取一致性

- [ ] 产出与声明的设计读取、三拨盘值一致（说了 VARIANCE 9 就不能交居中对称）
- [ ] 全页主题锁定：一页只有一个明暗主题，不混
- [ ] 能一句话说出这一页的**记忆点**是什么
- [ ] 动效气质与视觉气质一致（cohesion）

## C. 动效抽查（详表见 motion-craft.md）

- [ ] 无 `transition: all`；无 `ease-in`；无 `scale(0)` 入场
- [ ] UI 动效 ≤ 300ms；按压有 `:active` 反馈
- [ ] `prefers-reduced-motion` 兜底存在
- [ ] hover 动效包在 `@media (hover: hover) and (pointer: fine)` 内
- [ ] 只动了 transform/opacity

## D. 工程抽查（详表见 engineering-checklist.md）

- [ ] 图标按钮有 `aria-label`；表单控件有 label；焦点可见
- [ ] 弹窗：焦点陷阱 + Esc + 焦点返还；危险操作有确认/撤销
- [ ] 数字用 tabular-nums/等宽；标题 balance、正文 pretty
- [ ] 空态/加载态/错误态/成功态齐全；超长文本不破版
- [ ] 移动端：非对称布局塌缩单列；无 `h-screen`；无横向滚动条
- [ ] 暗色页有 `color-scheme` + `theme-color`

## D2. 中文页追加项（页面含中文时必查，详见 chinese-typography.md）

- [ ] 中文正文/UI 用系统字体栈；装饰中文字体只在创意标题且已 `text=` 子集化
- [ ] 中文行高 ≥ 1.5；正文 ≥ 14px；字重只用 400/500/600
- [ ] 中西文之间、中文与数字之间有空格；数字与单位之间有空格（°、% 除外）
- [ ] 全角标点；暗色页正文 `rgba(255,255,255,.85)` 且对比度 ≥ 7:1

## E. 三秒测试（最后一关）

眯起眼睛看整页三秒：

- [ ] 最重要的信息第一眼就在视线里吗？
- [ ] 页面在争夺注意力的元素 ≤ 1 个吗？
- [ ] 这一页**像不像 AI 做的**？像，就回到 A 节找出泄密点。
